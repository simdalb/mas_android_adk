
# ADMIN HOW-TO

This document is for the administrator of the project.

## Your recurring responsibilities
- keep account access secure
- maintain `.env`
- review release packets
- approve or deny internet requests
- keep platform accounts in good standing

## Secret handling
- use a password manager
- keep 2FA enabled
- never commit `.env`
- never commit signing keys
- back up account recovery codes safely

## Before release
- verify Play Console app listing
- verify privacy policy and support email
- verify ad disclosures
- verify subscription wording
- verify test and security notes in the release packet

## When approving internet access
Only approve if:
- the purpose is clear
- the requested access is necessary
- you understand the cost/risk tradeoff
